# NexGen Digital Marketing Portfolio Website

A modern, responsive portfolio website for **NexGen Digital**, a startup digital marketing agency.  
Showcases sample campaign strategies, capabilities, and services using a clean and trendy Next.js layout.

---

## 🚀 Features

- Fully built with **Next.js 13** (App Router) and **React**
- Modern, responsive design with **Tailwind CSS**
- **Portfolio / Case Studies** section showcasing marketing campaigns
- **Capabilities** section with metrics and achievements
- Modular **components** for easy scalability
- Interactive **UI elements** with subtle motion animations
- SEO-friendly structure and metadata

## ⚡ Getting Started (Local Development)

1. **Clone the repository**

```bash
git clone https://github.com/rachitrp2211/Digital_Marketing_WebSite.git
cd Digital_Marketing_WebSite
```
2. Install dependencies (using npm, yarn, or pnpm)
```
npm install
# or
pnpm install
# or
yarn
```
3.Run the development server
```
npm run dev
# or
pnpm dev
# or
yarn dev
```
Open http://localhost:3000 to view the site in your browser.

📦 Build for Production
```
npm run build
npm start
```
The site will be optimized for production.

🌐 Deployment
Recommended free deployment: Vercel
Connect your GitHub repo → Automatic Next.js support → Deploy in seconds
Optional: Connect a custom domain for a professional look

📝 Customization
Replace placeholder images in /public with real branding images.
Update content in app/ pages and components/ sections.
Modify Tailwind classes for color scheme, spacing, or layout changes.
